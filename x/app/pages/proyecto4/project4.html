Compilador

El siguiente documento sustenta todo el desarrollo del traductor para la gramática Vulcan. Este documento está divido en tres secciones: Análisis Léxico, Análisis Sintáctico y Análisis semántico. En cada sección se explicará el módulo con las clases correspondientes. Debemos precisar que este documento es un conglomerado de todos los documentos anteriores; sin embargo, está adaptado de tal forma que evita la redundancia. También es importante decir que el software cuenta con dos módulos, es decir, dos partes. Un analizador léxico y un analizador sintáctico; el semántico se encuentra incluido dentro del sintáctico.
